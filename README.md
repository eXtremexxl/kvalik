# Система авторизации транспортной компании

## Описание
WPF приложение для управления доступом пользователей с разделением ролей "Администратор" и "Пользователь".

## Функционал
### Авторизация
- Вход по логину/паролю
- Блокировка после 3 неудачных попыток
- Автоматическая блокировка при неактивности >30 дней

### Администратор
- Просмотр списка пользователей
- Добавление новых пользователей (логин, пароль, ФИО, email)
- Разблокировка аккаунтов

### Пользователь
- Обязательная смена пароля при первом входе
- Личный кабинет с информацией

## Технологии
- **Платформа:** .NET Framework 4.7.2, WPF
- **БД:** SQL Server, Entity Framework 6
- **Безопасность:** SHA256 хеширование паролей

## Быстрый старт
1. **Создайте БД:** выполните `Database/CreateDatabase.sql`
2. **Настройте подключение:** обновите строку в `App.config`
3. **Запустите:** соберите проект в Visual Studio

## Тестовые аккаунты
- **Администратор:** `admin` / `admin123`
- **Пользователь:** `user1` / `user123` 

## Безопасность
- Хеширование паролей (SHA256 + Base64)
- Защита от bruteforce
- Валидация всех входных данных
